# FreeStarGPT

FreeStarGPT 是一个基于 Vue 3 + TypeScript 开发的现代化 AI 聊天应用。它提供了一个简洁优雅的界面，支持与 AI 进行流式对话，并具有多会话管理、主题切换等功能。

## 特性

- 🚀 基于 Vue 3 + TypeScript + Vite 构建
- 💬 流式对话响应
- 📝 Markdown 支持，包括代码高亮
- 🎨 可自定义主题
- 📱 响应式设计
- 🔒 简单的密码保护
- 💾 本地会话存储
- 🌐 支持多会话管理

## 技术栈

- Vue 3 (Composition API + `<script setup>`)
- TypeScript
- Vite
- Vue Router
- Pinia
- Markdown-it


## 主要功能

### 聊天功能
- 支持流式对话响应
- Markdown 渲染
- 代码块语法高亮
- 消息复制功能
- 打字机效果

### 会话管理
- 创建新会话
- 切换会话
- 删除会话
- 会话本地持久化

### 主题系统
- 内置多个主题
- 支持自定义主题色
- 平滑过渡动画

### 安全特性
- 简单的密码保护
- 路由守卫
- 会话持久化
